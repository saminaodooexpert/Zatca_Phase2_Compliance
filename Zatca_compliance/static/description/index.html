Saudi ZATCA Phase 2 Compliance

Smart VAT Validation for Seamless Invoicing

Ensure your business stays compliant with Saudi Arabia's ZATCA Phase 2 regulations. This module automatically validates that a Customer VAT number is present before any invoice is posted, preventing legal and accounting errors.

Main Features

  • Automatic Validation: Checks for VAT numbers during the "Post" action.
  • Clear Error Messages: Guides users to fix missing data instantly.
  • Fully Compatible: Optimized for Odoo 17 Community and Enterprise.
